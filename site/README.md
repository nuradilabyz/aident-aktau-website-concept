# Aident — предварительная концепция

Статический одностраничный сайт на HTML, CSS и JavaScript. Концепт создан на основе открытых источников для Aident в Актау и не является официальным сайтом клиники.

## Запуск

Откройте `index.html` в браузере или запустите локальный статический сервер из папки `site`:

```sh
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Публикация

Для GitHub Pages укажите папку `site` как publish directory. После получения итогового URL добавьте абсолютный canonical URL и абсолютный `og:image` в `index.html`.

## Важные ограничения

- `noindex, nofollow` в meta-теге и `robots.txt` должны оставаться до согласования с владельцем.
- WhatsApp и график намеренно не включены: значения расходятся между 2GIS и Instagram.
- В концепте не используются фото пациентов, непроверенные врачи, цены, скидки или гарантии.
- Логотип: официальный Instagram-аватар, исходный URL и реестр — в `../audit/Aident-verification-report.xlsx`.
