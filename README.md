# Aident — website concept

Предварительная статическая концепция для Aident, цифровой стоматологии в Актау. До согласования с владельцем сайт остаётся неофициальным и закрыт от индексации.

## Структура

- `site/` — готовый статический сайт.
- `media/logo/` — исходный публичный Instagram-аватар клиники.
- `audit/` — XLSX-отчёт, доказательные скриншоты и генератор отчёта.
- `.github/workflows/deploy-pages.yml` — публикация папки `site/` в GitHub Pages.

## Локальный запуск

```sh
cd site
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Повторный деплой

1. Создайте публичный GitHub-репозиторий `aident-aktau-website-concept`.
2. В настройках Pages выберите источник `GitHub Actions`.
3. Опубликуйте ветку `main`; workflow развернёт содержимое `site/`.
4. После получения публичного HTTPS-адреса добавьте в `site/index.html` абсолютные `canonical` и `og:image`.
5. До согласования владельцем не удаляйте `noindex, nofollow`, `robots.txt` и пометку в футере.

## Контентные ограничения

В сайт намеренно не включены спорные WhatsApp и график: они различаются между 2GIS и Instagram. Не использованы цены, акции, врачи, гарантии, отзывы пациентов, фото пациентов и сторонние стоковые материалы.

Подробный реестр данных, источников и проверки: `audit/Aident-verification-report.xlsx`.
