# Aident — website concept

Предварительная статическая концепция для Aident, цифровой стоматологии в Актау. До согласования с владельцем сайт остаётся неофициальным и закрыт от индексации.

Публичная проверяемая версия: <https://nuradilabyz.github.io/aident-aktau-website-concept/>.

## Структура

- `site/` — готовый статический сайт.
- `media/logo/` — исходный публичный Instagram-аватар клиники.
- `audit/` — XLSX-отчёт, доказательные скриншоты и генератор отчёта.
- `.github/workflows/deploy-pages.yml` — публикация папки `site/` в GitHub Pages.

## Локальный запуск

```sh
cd site
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Повторный деплой

1. В настройках GitHub Pages выберите источник `GitHub Actions`.
2. Опубликуйте изменения в ветке `main`; workflow развернёт содержимое `site/`.
3. Проверьте HTTPS-версию и ссылки перед тем, как сообщать о новом деплое.
4. До согласования владельцем не удаляйте `noindex, nofollow`, `robots.txt` и пометку в футере.

## Контентные ограничения

Основная запись в этой версии ведёт на WhatsApp из карточки 2GIS: `wa.me/77758255243`. В bio Instagram указан другой, неполный номер; его нужно подтвердить у владельца до финального запуска. График не опубликован из‑за расхождения источников. Не использованы цены, акции, врачи, гарантии, фото пациентов и сторонние стоковые материалы. В блоке отзывов — только два коротких публичных фрагмента из 2GIS со ссылкой на первоисточник.

Подробный реестр данных, источников и проверки: `audit/Aident-verification-report.xlsx`.
